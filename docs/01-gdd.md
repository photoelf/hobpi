# 01. Game Design Document (ядро)

## 1. Игровые лупы

### 1.1. Микро-луп: Бой (2–4 минуты)

```
Расстановка → [Раунд: очередь по инициативе → ход стека (движение/атака/выстрел/защита/ожидание)
              → приём героя (1 раз в раунд) → проверка морали/удачи] → Победа/Поражение → Трофеи
```

### 1.2. Мид-луп: Сессия (5–10 минут)

```
Вход → Собрать доход с точек → Нанять прирост бойцов → Потратить: улучшить точку / здание
     → Пойти в бой: PvE «двор» (интерактивно) ИЛИ PvP «разборка» (асинхронно)
     → Опыт герою → выбор навыка → Выход
```

### 1.3. Макро-луп: Неделя (7 дней)

```
Понедельник: недельное событие («Неделя качков», «Неделя мажоров», «Облава»)
           → сброс прироста бойцов, обнуление сезонных лимитов PvP
Пн–Вс:       набор рейтинга, клановые войны за ключевые точки
Воскресенье: подсчёт рейтинга кланов, выплата с «Серых схем», ротация лидерборда
```

## 2. Сущности

| Сущность | Описание |
|---|---|
| **Игрок** | Один Telegram-аккаунт = один профиль. Ранг, рейтинг, клан. |
| **Герой** | Аватар игрока. Класс, уровень, 4 первичных стата, навыки, приёмы, слоты артефактов. |
| **Бригада (армия)** | До 7 стеков юнитов, каждый стек = тип юнита + количество. |
| **База клана** | «Замок»: здания найма, экономические и спец-здания. |
| **Точка** | Источник дохода (киоск, парковка, автосервис, клуб, рынок). Уровни 1–3. |
| **Район** | Узел на карте города. Даёт бонусы, может быть захвачен кланом. |
| **Клан** | Объединение игроков. Общая казна, клановые войны, рейтинг. |

## 3. Первичные статы героя

Прямой перенос HoMM3 (Attack / Defense / Spell Power / Knowledge) в сеттинг:

| Стат | Русское имя | Эффект |
|---|---|---|
| Attack | **Сила** | +5% урона бригады за очко (мультипликативно с разницей А/З стеков) |
| Defense | **Броня** | −2.5% входящего урона за очко |
| Spell Power | **Авторитет** | Множитель силы приёмов героя |
| Knowledge | **Связи** | Запас «связей» (мана) = Связи × 10 |

## 4. Статы юнита

| Стат | Имя | Комментарий |
|---|---|---|
| attack | Атака | Влияет на урон через разницу с защитой цели |
| defense | Защита | |
| minDmg / maxDmg | Урон | Урон одной единицы стека |
| hp | Живучесть | ХП одной единицы |
| speed | Ход | Дальность движения в клетках + позиция в очереди инициативы |
| initiative | Инициатива | Тай-брейк очереди (по умолчанию = speed) |
| shots | Патроны | Для стрелков; 0 = ближник |
| growth | Прирост | Сколько единиц в неделю даёт здание найма |
| cost | Цена | Ресурсы за 1 единицу |
| tier | Тир | 1–7 |
| traits | Свойства | Набор флагов-способностей (см. [03-combat.md](03-combat.md)) |

## 5. Прогрессия

### 5.1. Уровни героя

XP-кривая: `xp(L) = 1000 * L^1.5`. На каждом уровне:
- +1 к случайному первичному стату (веса зависят от класса),
- каждый 2-й уровень — выбор из 3 навыков (skill tree, см. [05-heroes.md](05-heroes.md)).

### 5.2. Ранги игрока (влияют на матчмейкинг и доступ к контенту)

| Ранг | Название | Требование |
|---|---|---|
| 1 | Пацан с района | старт |
| 2 | Бегунок | 3 победы PvP |
| 3 | Бригадир | ур. 5 |
| 4 | Смотрящий | ур. 10 + 1 захваченный район |
| 5 | Авторитет | ур. 15 + рейтинг 1200 |
| 6 | Вор в законе | ур. 20 + рейтинг 1500 |
| 7 | Хозяин города | топ-10 сезона |

### 5.3. Ограничители темпа (anti-grind)

- **Топливо** (энергия): 10 единиц, +1 каждые 12 минут. PvE-бой = 1, PvP-разборка = 2.
- Прирост бойцов пополняется раз в 24 часа (не «за просмотр рекламы»).
- Доход с точек капает 4 раза в сутки, казна имеет потолок вместимости (стимул заходить).

## 6. Экраны мини-аппа

| Экран | Содержимое | Приоритет |
|---|---|---|
| **Онбординг** | Выбор фракции → выбор класса героя → кличка → туториал-бой | MVP |
| **База** | Ресурсы, здания, найм бойцов, улучшения | MVP |
| **Бригада** | Состав армии, перестановка стеков, роспуск | MVP |
| **Герой** | Статы, навыки, «бумажная кукла» артефактов, приёмы | MVP |
| **Район (карта)** | Список PvE-точек и «дворов» с нейтралами | MVP (список, не карта) |
| **Арена** | Список соперников PvP, история боёв, реплеи | MVP |
| **Бой** | Сетка 12×9, очередь инициативы, панель действий | MVP |
| **Рейтинг** | Топ игроков, топ кланов | MVP |
| **Клан** | Казна, состав, войны | v0.3 |
| **Карта города** | Настоящая карта районов с захватами | v0.4 |

Навигация — нижний таб-бар из 5 пунктов: **База / Бригада / Район / Арена / Топ**.

## 7. Онбординг (первые 3 минуты)

1. Экран выбора фракции: 4 карточки (в MVP), у каждой одна фраза-хук и иконка стиля.
2. Выбор класса героя: 3 карточки, отличаются стартовыми статами и стартовым приёмом.
3. Кличка (по умолчанию — TG first_name).
4. Обучающий бой: 2 своих стека против 2 нейтральных «алкашей у ларька». Подсказки поверх сетки.
5. Первая награда: 500 налика + 1 точка «Ларёк» бесплатно.
6. CTA: «Найми бойцов» → «Иди на разборку».

## 8. Нарратив и кампании (после MVP)

- **«Война за набережную»** — Старые авторитеты vs Мажоры за элитный район.
- **«Болота и промзона»** — Гопники и Рыночные vs Похоронка за серую логистику.
- **«Тихие башни»** — Айтишники и Подполье переводят город на безнал.

Формат: цепочка из 7–10 боёв с фиксированными армиями и текстовыми вставками между ними,
раскрывает механики одной фракции. Служит и туториалом второго уровня, и лорным контентом.

## 9. Баланс: базовые принципы

1. **Сила бригады** измеряется в AI-очках (`ai_value`), как в HoMM3. Матчмейкинг PvP и генерация
   PvE-нейтралов работают по этому числу, а не по уровню героя.
2. Юнит тира N примерно вдвое дороже и сильнее юнита тира N−1, но прирост в неделю падает
   быстрее — топовые тиры дефицитны.
3. Стрелки сильнее в открытом поле, ближники — при быстром сближении. Каждая фракция имеет
   чёткий перекос (см. [02-factions.md](02-factions.md)), контр-пики существуют.

   Измеренное состояние баланса (`node tools/balance.ts 400`, 4800 автобоёв на равный
   бюджет найма, средняя длина боя 8.8 раунда):

   | Фракция | Винрейт | vs Авторитеты | vs Качалка | vs Гопники | vs Айтишники |
   |---|---|---|---|---|---|
   | Старые авторитеты | 56.5% | — | 10% | 82% | 90% |
   | Качалка | 48.3% | 98% | — | 13% | 25% |
   | Гопники | 48.6% | 2% | 72% | — | 58% |
   | Айтишники | 46.5% | 43% | 64% | 45% | — |

   Все в коридоре 40–60%, при этом попарные матчапы резко несимметричны: Авторитеты бьют
   Гопников, Гопники бьют Качалку, Качалка бьёт Авторитетов. Это **намеренно** — треугольник
   контр-пиков делает выбор фракции соперника осмысленным решением, а не косметикой.
   Балансировка автоматизирована: `tools/autotune.ts` ищет масштаб урона линеек,
   `tools/balance.ts` проверяет результат.
4. Герой — усилитель, а не самостоятельная боевая единица: он не имеет ХП и не может быть убит.
5. Приёмы стоят «связей», запас которых конечен на бой → цена решения «когда кастовать».
