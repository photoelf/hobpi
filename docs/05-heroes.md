# 05. Герои, навыки, артефакты

## 1. Классы героев

Как в HoMM3, деление на «might» (Сила/Броня) и «magic» (Авторитет/Связи), но без жёсткой привязки
класса к фракции: любой класс можно взять за любую фракцию (упрощение для мобильного онбординга).

| Класс | Тип | Старт: Сила / Броня / Авторитет / Связи | Специализация | Стартовый приём |
|---|---|---|---|---|
| **Авторитет** | might | 4 / 3 / 1 / 1 | +5% урона всей бригады | Общий сбор |
| **Смотрящий** | гибрид | 2 / 3 / 2 / 2 | +15% дохода со всех точек | Запугивание |
| **Качок-тренер** | might | 3 / 4 / 1 / 1 | Юниты T1–T3 получают +2 к атаке | Адреналин |
| **Мажор-бизнесмен** | magic | 1 / 2 / 3 / 4 | −20% стоимости найма | Санитары |
| **Опер** | magic | 2 / 3 / 3 / 2 | Приёмы-контроль стоят −30% связей | Облава |
| **Диджей** | magic | 2 / 2 / 3 / 3 | +1 к удаче и морали бригады | Адреналин |

✅ В MVP: **Авторитет**, **Опер**, **Мажор-бизнесмен** (по одному на архетип). Остальные — v0.3.

## 2. Прокачка

- XP за бои: `xp = сумма(ai_value убитых стеков) / 10`, за PvP-победу ×1.5.
- Кривая: `xp_нужно(L) = 1000 * L^1.5`.
- Каждый уровень: +1 к стату по весам класса (might-классы чаще получают Силу/Броню).
- Каждый 2-й уровень: выбор 1 навыка из 3 случайных доступных.

## 3. Навыки (аналог вторичных навыков HoMM3)

Каждый навык имеет 3 уровня: Базовый / Продвинутый / Мастер.

### Боевые
| Навык | Эффект (Б / П / М) |
|---|---|
| **Тактика** | Расширение зоны расстановки на 1 / 2 / 3 колонки |
| **Стрельба** | +10 / 25 / 50% урона стрелков |
| **Кулаки** | +10 / 25 / 50% урона ближников |
| **Дисциплина** | +1 / +2 / +3 к морали |
| **Фарт** | +1 / +2 / +3 к удаче |
| **Скорость** | +1 / +1 / +2 к скорости всех стеков |
| **Первая помощь** | В конце боя восстанавливается 10 / 25 / 40% павших T1–T2 |

### Экономические
| Навык | Эффект |
|---|---|
| **Коммерция** | +10 / 20 / 35% дохода налика |
| **Логистика** | +1 / +2 / +3 к максимуму топлива |
| **Кадры** | +15 / 30 / 50% к приросту бойцов |
| **Дипломатия** | Шанс 10 / 20 / 35% что нейтральный «двор» присоединится вместо боя |

### «Магические» (связи)
| Навык | Эффект |
|---|---|
| **Авторитет** | +25 / 50 / 100% к силе приёмов |
| **Связи** | +25 / 50 / 100% к запасу связей в бою |
| **Разведка** | Видно точный состав армии противника до боя |
| **Прессинг** | Приёмы-дебаффы длятся +1 / +2 / +3 раунда |

## 4. Артефакты («бумажная кукла»)

Слоты (HoMM3-style, 8 слотов в MVP):

```
        ┌────────┐
        │ ГОЛОВА │        кепка, очки, шапка
┌───────┼────────┼───────┐
│ ШЕЯ   │ ТОРС   │ РУКИ  │  цепь / спортивка / кастет, барсетка
└───────┼────────┼───────┘
        │  НОГИ  │        кроссы, берцы
        ├────────┤
        │ СТВОЛ  │        бита, нож, ТТ, обрез
        ├────────┤
        │ ТАЧКА  │        мерин, бэха, девятка
        ├────────┤
        │ ТЕЛЕФОН│        сименс, нокиа, айфон
        └────────┘
```

### Редкости
| Редкость | Цвет | Примерный бюджет статов |
|---|---|---|
| Обычный | серый | 2 очка |
| Ценный | зелёный | 4 очка |
| Крутой | синий | 7 очков |
| Реликвия | оранжевый | 12 очков + уникальный эффект |

Очко = +1 к первичному стату, или +1 морали/удачи, или +5% к профильному проценту.

### Примеры (реализованы в MVP)

| Артефакт | Слот | Ред. | Эффект |
|---|---|---|---|
| Спортивка «три полоски» | Торс | Обычный | +1 Сила, +1 мораль |
| Золотая цепь | Шея | Ценный | +2 Авторитет, +1 влияния за победу |
| Очки Ray-Ban | Голова | Ценный | +2 удача |
| Кроссы «белые» | Ноги | Обычный | +1 к скорости всех стеков |
| Кастет | Руки | Ценный | +2 Сила |
| Барсетка | Руки | Ценный | +10% дохода налика |
| Пейджер | Телефон | Обычный | +5 связей |
| Nokia 8800 | Телефон | Крутой | +15 связей, приёмы −20% стоимости |
| Лупатый мерин | Тачка | Крутой | +2 к максимуму топлива, +1 скорости бригады |
| Бэха «пятёрка» | Тачка | Реликвия | +2 Сила, +2 Броня, +1 мораль, +1 удача |
| Кепка «Адидас» | Голова | Обычный | +1 Броня |
| Часы «Ролекс» | Шея | Крутой | +2 Авторитет, +20% дохода |
| Берцы | Ноги | Ценный | +2 Броня |
| Печатка | Руки | Крутой | +1 Сила, +1 Авторитет, +1 мораль |
| Бита «Аргумент» | Ствол | Обычный | +2 Сила |
| Нож-бабочка | Ствол | Ценный | +2 Сила, +2 удача |
| ТТ | Ствол | Крутой | +4 Сила, +2 Авторитет, +1 мораль |
| Обрез | Ствол | Реликвия | +5 Сила, +2 Броня, +2 мораль, +3 удача |

Стволы — вымышленные и безмарочные, эффект чисто числовой (модификаторы статов героя).
Отдельной механики огнестрела в бою нет: стрельба — свойство юнитов, а не героя.

### Комбо-артефакты (v0.3)

| Комплект | Состав | Бонус комплекта |
|---|---|---|
| **Комплект мажора** | Часы «Ролекс» + Nokia 8800 + Бэха + костюм | +50% дохода, +2 ко всем статам, приём «Решить вопрос» (мгновенная победа над нейтралом ≤ 30% силы) |
| **Комплект гопника** | Спортивка + кепка + кроссы + цепь | +3 морали, +25% урона T1–T3 |
| **Комплект качка** | Кастет + берцы + пояс + протеин | +4 Сила, ближники получают «двойной удар» на первом ходу |

### Источники артефактов

- PvE-«дворы» (шанс дропа зависит от силы нейтралов),
- недельные квесты,
- PvP-сезонные награды,
- «Барахолка» (обмен между игроками, v0.4).
