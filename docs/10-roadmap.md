# 10. Роадмап

Оценки — в неделях работы одного разработчика с ИИ-ассистентом.

## v0.1 — MVP (сделано)
Движок, 4 фракции, экономика, PvE, асинхронный PvP, 6 экранов, реплеи.

## v0.2 — Живой тест (2 недели)
- [ ] Реальный Telegram-бот: вебхук, пуши, инлайн-шаринг реплеев, deep-links.
- [ ] Телеметрия: воронка онбординга, длина сессии, распределение исходов боёв.
- [ ] Балансный автопрогон: 100 000 симуляций «армия A vs армия B» на всех парах фракций,
      поиск доминирующих стратегий.
- [ ] Туториал-бой с подсказками поверх сетки.
- [ ] Настройки уведомлений.

## v0.3 — Глубина (4 недели)
- [ ] Фракции **Похоронка** (Necropolis) и **Подполье** (Dungeon).
- [ ] Улучшенные версии бойцов (второй уровень зданий найма).
- [ ] Оставшиеся 3 класса героя, полное дерево навыков (15 навыков × 3 уровня).
- [ ] Комбо-артефакты.
- [ ] Кланы: создание, казна, состав, чат-интеграция.
- [ ] Ежедневные и недельные квесты.

## v0.4 — Мета и социалка (5 недель)
- [ ] Фракции **Рыночные** (Fortress) и **Творческая тусовка** (Rampart).
- [ ] Карта города: 12 районов, захват точек кланами, клановые войны.
- [ ] Препятствия и особенности местности на поле боя.
- [ ] Барахолка (обмен ресурсов и артефактов) + античит на мультиаккинг.
- [ ] Сезоны: 4-недельный цикл с обнулением рейтинга и сезонными наградами.

## v0.5 — Контент и монетизация (5 недель)
- [ ] Фракция **Мажоры** (Conflux).
- [ ] 3 сюжетные кампании по 8–10 боёв.
- [ ] Telegram Stars: ускорения, слоты, косметика, Battle Pass.
- [ ] Косметика героя (визуал без статов).
- [ ] Локализация: EN (сеттинг адаптируется как «90s Eastern European crime»).

## v1.0 — Публичный запуск
- [ ] Все 9 фракций (63 бойца) + улучшения.
- [ ] Миграция на PostgreSQL, горизонтальное масштабирование API.
- [ ] Арт-пасс: единый визуальный стиль, спрайты бойцов, анимации боя.
- [ ] Живые турниры и события с призовым фондом.
- [ ] Публичное API реплеев (виджет-встраивание боёв в веб).

## Технический долг, который придётся отдать

| Долг | Когда платить |
|---|---|
| `node:sqlite` → PostgreSQL | при ~5 000 DAU или при появлении второй ноды API |
| Реплеи в таблице → объектное хранилище | при > 1 млн боёв |
| Отсутствие миграционного инструмента (схема создаётся `CREATE TABLE IF NOT EXISTS`) | перед первым релизом с живыми данными — ввести версионированные миграции |
| Баланс, зашитый в TS-константы | v0.3 — вынести в JSON/БД, чтобы править без деплоя |
| ИИ — жадная эвристика без просчёта на ход вперёд | v0.4, когда фракций станет 7+ и эвристика начнёт откровенно тупить |
| Нет e2e-тестов клиента | v0.2 — Playwright на ключевой воронке |
